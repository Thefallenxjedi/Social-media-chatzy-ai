# Chatzy AI Platform Features & Pricing

This document outlines the core capabilities, feature sets, and pricing tiers for the Chatzy AI platform. This can be used as a reference for writing feature-specific or lead-generation LinkedIn posts.

## Pricing Tiers
- **Starter:** 3,499 / month
- **Growth:** 7,499 / month
- **Advanced:** 18,499 / month
- **Enterprise:** Custom Pricing

---

## Core Features & Capabilities

### 1. Channels & Deployment
* **Omnichannel Support:** WhatsApp, Website Widget, Voice, Instagram DMs, Instagram Comments, LinkedIn DMs, In-app integration, Messenger, and Telegram.
* **Deployment Limits:** Starter includes any 2 channels. Growth and Advanced include all 6 unified channels.
* **White-labeling:** Option to remove the Chatzy watermark from the website widget.

### 2. Types of AI Agents
* **Rule-Based:** Flow-based, no-code agents.
* **Conversational:** Natural language, AI-driven agents trained on knowledge bases.
* **Voice AI:** Voice-based conversational agents.

### 3. Multi-Agent & Team Collaboration
* **Multi-Agentic Squads:** Deploy multiple specialized agents that work together.
* **Shared Inbox:** A unified omnichannel inbox for the entire team.
* **Human Handoff:** Instant transition from AI to human agents.
* **Auto-Routing:** Chat auto-assignment and routing across AI and human agents.
* **Voice Integration:** AI + Human WhatsApp calling directly from the Chatzy inbox.
* **Access Control:** Custom roles and permissions for team members.

### 4. Automation & Journeys
* **Agentic Journeys:** End-to-end customer lifecycle automation.
* **Advanced Workflow Builder:** Includes branching logic flows and API calls inside flows.
* **Campaigns:** Run broadcast campaigns and manage audiences using custom attributes and tags.
* **Social Automation:** Instagram comment automation and LinkedIn outbound sequences.

### 5. APIs, Webhooks & Extensibility
* **Chatzy AI APIs:** API access to embed the AI agent directly into your own app.
* **WhatsApp APIs:** Full access to WhatsApp APIs and advanced webhooks.
* **Custom LLMs:** Ability to add your own LLM API keys (ideal for startups with AI program credits).
* **Tracking:** CAPI (Meta Conversions API) setup assistance and custom list triggers.

### 6. Analytics & Reporting
* **Basic & Advanced Analytics:** Track agent performance, resolution rates, and conversation metrics.
* **Custom Reporting:** Custom reports via Redash for enterprise clients.

### 7. Support & Onboarding
* **Starter:** Fully self-serve using video tutorials, documentation, and email/live chat support.
* **Growth:** Dedicated WhatsApp chat support, WhatsApp setup assistance (Official API, Blue Tick verification help), and CTWA/CAPI setup assistance.
* **Advanced/Enterprise:** Dedicated Customer Success/Account Manager, priority white-glove onboarding (live in days, not weeks), and special customizations.
